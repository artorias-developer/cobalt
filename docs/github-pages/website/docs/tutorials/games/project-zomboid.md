# Project Zomboid

This tutorial explains how to set up a Project Zomboid server in the Cobalt dashboard.

## Changing configs

1. Open the servers page.

2. Find the server you want to change settings in the table and click the first button in the actions column.

3. Click the stop button in the control block.

4. Open the files tab at the top.

   ---

   ### `serverconfig.ini`

   1. Open the `Zomboid/Server/serverconfig.ini` file in the file editor.

   2. Update the settings.

## Adding mods

1. Open the servers page.

2. Find the server you want to add mods in the table and click the first button in the actions column.

3. Click the stop button in the control block.

4. Upload mods using one of the following methods:

::::details File manager
1. Open the files tab at the top.

2. Open the `Zomboid/Server/serverconfig.ini` file in the file editor.

3. Update the server settings:

```ini
# Set the mod IDs separated by ;
Mods=VanillaGearExpanded;mapmodmanager

# Set the Steam Workshop IDs separated by ;
WorkshopItems=3401134276;3360392628
```

:::tip
You can find mods on [Steam Workshop](https://steamcommunity.com/app/108600/workshop/).

Example: https://steamcommunity.com/sharedfiles/filedetails/?id=3401134276

In this case, the ID is `3401134276`.
:::

:::tip
Authors also often include the mod ID and Steam Workshop ID at the very bottom of the description.
:::

4. Click the save button in the bottom right corner.
::::

5. Open the overview tab at the top.

6. Click the restart button in the control block.
