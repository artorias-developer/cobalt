# Terraria

This tutorial explains how to set up a Terraria server in the Cobalt dashboard.

## Changing configs

1. Open the servers page.

2. Find the server you want to change settings in the table and click the first button in the actions column.

3. Open the files tab at the top.

   ---

   ### `serverconfig.cfg`

   1. Open the `serverconfig.cfg` file in the file editor.

   2. Update the settings.

## Adding mods

:::warning
Mods can only be added for the `tModLoader` loader.
:::

1. Open the servers page.

2. Find the server you want to add mods in the table and click the first button in the actions column.

3. Upload mods using one of the following methods:

::::details File manager
1. Open the files tab at the top.

2. Open the `data/Mods` folder in the file manager.

3. Click the upload button in the bottom left corner.

4. Select the mods files from your device.

:::tip
You can find mods on [Steam Workshop](https://steamcommunity.com/app/1281930/workshop/).
:::

5. Click the upload button in the bottom right corner.
::::

4. Open the files tab at the top.

5. Open the `data/Mods/enabled.json` file in the file editor.

6. Add the names of the mod files to the list, separating them with commas:

```json
[
   "ModFileName1",
   "ModFileName2"
]
```

7. Click the save button in the bottom left corner.

8. Open the overview tab at the top.

9. Click the restart button in the control block.
