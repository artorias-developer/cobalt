# Users

This tutorial explains how to create and update users in the Cobalt dashboard.

## Creating a user

1. Open the users page.

2. Click the create button in the bottom left corner.

3. Enter the user's login, password and select the role.

4. Click the create button in the bottom right corner.

## Updating a user

1. Open the users page.

2. Find the user you want to update in the table and click the first button in the actions column.

3. Update the user's login, password or role.

4. Click the save button in the bottom right corner.
