# Roles

This tutorial explains how to create and update roles in the Cobalt dashboard.

## Creating a role

1. Open the roles page.

2. Click the create button in the bottom left corner.

3. Enter the role name and select the permissions you want to assign to it.

:::tip
You can select or deselect all items in a group by clicking on its title.
:::

:::info
The access can be restricted by role for virtually every section of the dashboard.
:::

4. Click the create button in the bottom right corner.

## Updating a role

1. Open the roles page.

2. Find the role you want to update in the table and click the first button in the actions column.

3. Update the role name or permissions you want to assign to it.

:::tip
You can select or deselect all items in a group by clicking on its title.
:::

4. Click the save button in the bottom right corner.

:::info
When you change role, all users with that role receive updates immediately.
:::
