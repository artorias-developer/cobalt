# Servers

This tutorial explains how to create servers in the Cobalt dashboard.

## Creating a server

1. Open the servers page.

2. Click the create button in the bottom left corner.

3. Choose the game from the list.

4. Click the next button in the bottom right corner.

5. Enter the server's name, select the loader and version. 

6. Click the create button in the bottom right corner.

:::tip
You don't need to refresh the page to check for changes in the server's status. It will update automatically as soon as the server is created, and you'll see the server control buttons.
:::
